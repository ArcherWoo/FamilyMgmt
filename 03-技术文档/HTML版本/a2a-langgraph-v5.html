🏗️ 架构工作流程

1. A2A Protocol Layer

Google Agent2Agent 通信协议层，负责Agent身份认证、消息路由、安全通信和协议标准化

2. LangGraph Orchestrator

状态机驱动的工作流编排引擎，提供状态管理、条件路由、并行执行和错误恢复

3. AI Agent Layer

智能体层：饮食规划Agent、运动推荐Agent、穿搭建议Agent，每个Agent都有独立的A2A身份

4. Tool Execution Agents

工具执行Agent集群：天气服务、购物服务、健康数据、图像识别等分布式工具服务

5. Unified State Management

统一状态管理层：LangGraph State + A2A Session Management + 持久化存储
