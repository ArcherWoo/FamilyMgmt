🎯 MCP协议概述

Model Context Protocol (MCP) 是由Anthropic主导开发的开源协议，专门用于AI模型与外部数据源和工具的标准化连接。在A2A + MCP双引擎架构中，MCP负责外部服务集成，与A2A协议形成完美互补。

⚡ 双引擎协同
