### 必需工具

- VS Code

## 📋 开发规范

### 代码规范
- 使用ESLint进行代码检查
- 遵循JavaScript Standard Style
- 统一的缩进和命名规范

### Git规范
- 使用Git Flow工作流
- 提交信息格式：`type(scope): description`
- 定期合并和清理分支

### 测试规范
- 单元测试覆盖率 > 80%
- 集成测试必须通过
- 性能测试达标
